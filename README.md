# Складова система

## Описание

Програмата позволява:
- Добавяне на продукти
- Преглед на продукти
- Търсене на продукт
- Продажба на продукт
- Изтриване на продукт
- Запис и зареждане на данни от JSON файл

## Файлове
- main.py -> Основна програма и меню
- models.py -> Клас Product
- data.json -> Запазени продукти
- requirements.txt -> Използвани библиотеки
